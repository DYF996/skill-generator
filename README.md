# skill-generator

将重复性工作自动生成可复用 Skill 的工具。支持三种工作模式：直接执行、先计划、生成技能。

## 功能

### 三种工作模式

| 模式 | 命令 | 说明 |
|------|------|------|
| **直接执行** | `直接执行 需求` 或 `做 需求` | 立刻动手，不输出计划 |
| **先计划** | `先计划 需求` 或 `计划 需求` | 先列步骤清单，你确认后再执行 |
| **生成技能** | `生成技能 需求` | 输出完整 Skill 包，固化工作流 |

**自动判断**：不带命令时，系统会根据你的话自动判断走哪个模式。

### 简化/详细模式

- 说"简化模式" → 用大白话输出，不秀技术细节
- 说"详细模式" → 恢复技术清单

### 重复性任务检测

完成任务后，自动判断这活儿以后是否还会重复做。如果是，主动问你是否要生成 Skill 以便下次复用。

## 快速开始

1. 将 `SKILL.md` 放入你的 OpenClaw skills 目录：
   ```
   ~/.qclaw/skills/skill-generator/SKILL.md
   ```

2. 可选：添加 `references/` 目录获取更详细的规范文档

## 目录结构

```
skill-generator/
├── SKILL.md                    # 主技能文件
└── references/                 # 参考文档（可选）
    ├── skill_spec.md          # Skill 生成规范
    └── interactive_protocol.md # 交互式求解协议
```

## 使用示例

### 示例 1：直接执行

```
> 做 帮我整理桌面的文件
```

### 示例 2：先计划

```
> 计划 每周生成销售报告
```

### 示例 3：生成技能

```
> 生成技能 写周报
```
